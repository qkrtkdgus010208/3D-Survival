# ⛺3D Survival⛺

## 📜 게임 개요
- **3D Survival**은 사용자가 캐릭터를 조작하며 맵에서 상호작용 할 수 있는 **3D 생존 게임**입니다.
- 플레이어는 맵을 탐색하며 **NPC와 싸우고, 자원을 캐고, 아이템을 줍고, 플랫폼을 밟고 이동**할 수 있습니다.
- 다양한 아이템과 무기를 활용하면 **생존에 도움이 될 수 있습니다.**
- 수단과 방법을 가리지 않고 **생존하세요!**

---

## 🕹️ 조작 방법
| 조작키 | 기능 |
|------|------|
| `WASD` / `←↑↓→` | 캐릭터 이동 |
| `마우스 좌클릭` | 공격 |
| `TAB` | 인벤토리 |
| `E` | 아이템 획득 |
| `V` | 시점 변경 |

---

## 🎮 주요 기능

### 1️⃣ 캐릭터 이동 및 맵 탐색
- **WASD or 키보드 화살표 키**로 이동 가능

### 2️⃣ 사용자 입력 처리
- **Input System과 PlayerInput**을 이용한 사용자 입력 처리
- **Invoke Unity Events** 방식으로 이벤트 연결

### 3️⃣ 생존 시스템 설계
- **체력, 허기, 스테미나** 보유
- **허기가 0이 되면 체력이 감소**되는 시스템

### 3️⃣ 낮과 밤 사이클 설계
- **Skybox Material**로 빛을 제작
- **AnimationCurve**를 활용해 낮과 밤을 구현

### 4️⃣ 상호작용
- **E키를 통해 아이템 습득** 가능
- **십자선을 통해 아이템 정보 표시**

### 5️⃣ 인벤토리
- **획득한 아이템 정보 표시**
- **아이템 타입에 맞는 버튼 표시**
- **슬롯 당 12개 보유 가능** (소비 아이템)

### 6️⃣ 스크립터블 오브젝트
- **ScriptableObject**를 이용해 **Item Data**를 제작
- **enum**을 활용한 아이템 타입 설정
  
### 7️⃣ NPC
- **Idle, Wander, Attack 상태** 보유
- **플레이어 탐지 X : Idle <=> Wander**
- **플레이어 탐지 O : Attack**

### 8️⃣ 음악 시스템
- **뮤직 존**을 적용해 **플레이어가 특정 공간에 진입**하면 음악이 나옴
- **FadeIn, FadeOut 효과**를 통해 몰입도 향상
- **발걸음 소리**를 통해 생동감 부여

### 9️⃣ 플랫폼
- **일정 시간이 지나면 플레이어를 위로 날려버리는 플랫폼**
- **플레이어를 태우고 이동하는 플랫폼**

### 🔟 레이저 트랩
- **Line Renderer**를 이용해서 레이저 표시
- **Raycast**를 이용해 1초 마다 데미지 부여
